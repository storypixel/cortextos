# Research Agent Tuning Knobs

Use this checklist during `research-agent-setup` and later quality reviews.

## Identity

- agent name
- timezone
- preferred research style
- brief length and tone
- update frequency

## Research Scope

- target audience
- niche, market, product, or community
- repeated questions the agent should answer
- source categories to include
- topics and sources to exclude

## Sources

- YouTube channels
- Reddit communities
- GitHub search queries
- GitHub trending filters
- Hacker News enabled/disabled
- arXiv queries
- RSS feeds
- custom URLs
- optional social search via Apify

## Scoring

- niche keywords
- tutorial/how-to keywords
- source-type bonuses
- negative keywords
- minimum score threshold
- top N per run
- freshness window
- suppress recently delivered window

## Delivery

- local markdown only
- Telegram
- Slack
- dashboard or another local path
- approval required before external delivery
- safe autonomous delivery exceptions, if any

## Schedules

- daily research brief time
- topic briefing time
- weekly trends review
- weekly quality review
- quiet hours
